このシナリオはダミーです。

フラグ処理の作例を紹介します。
「カギを入手してドアを開ける」という、よくある
シチュエーションのシナリオを作ってみます。

*Main
君は自宅の玄関前にいてドアにはカギがかかっている。
1.ドアを開ける
2.ポストを調べる
@SEL Door
@SEL Post
@INP

*Door
@IF FLG 0 0 Nokey
カギでドアを開け、家に入れました。
よかった、よかった。
@INP
@END

*NoKey
カギが無いと開きません。
@JMP Main

*Post
@IF FLG 0 0 Getkey
何もありません。
@JMP Main

*GetKey
ポストにカギがありました！
@JMP Main
